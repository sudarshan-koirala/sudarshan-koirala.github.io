KIRO is an AI-powered code editor made by Amazon. Think of it like having a very smart assistant that writes code for you, just by describing what you want in plain English.

## What Makes KIRO Different?

The big difference from other tools: KIRO makes a **plan** before writing any code. Most AI tools just write code immediately. KIRO first asks: what should we build? How should it work? Then it codes.

This approach leads to more structured, maintainable code from the start.
